Object Oriented Programming
1. Class and Object
   CLASS --> A class is a user-defined blueprint that groups 
   related data (attributes) and behavior (methods) into a single
   unit, from which objects are created.

   OBJECT --> An object is a real instance of a class. 
   It is created from a class and contains its own data 
   (attributes) and can perform the behaviors (methods) 
   defined in that class.

2. Class has data/attribute and function/methods.
   Attributes called itself, and methods are called manually.
